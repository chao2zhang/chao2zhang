# Blog Post Design: APM for Software Engineers

## Approved Structure

1. Cold open (~200w) - StarCraft/agent parallel moment
2. Reluctant convert arc (~500w) - Cursor -> Claude -> Gastown -> corporate mandate
3. Comparison table (~100w) - expanded chart
4. Deep dives into parallels (~800w) - APM, macro/micro, fog of war, supply
5. Corporate plot twist (~400w) - hobby becomes KPI
6. Closing (~200w) - ceiling is strategy, not mechanics

## Tone
- HN voice, self-deprecating, no marketing speak
- Real tool names, real frustrations
- Honest about failures
